# Sample MCP Apps

MCP Appsのサンプルプロジェクトです。

詳細は以下の記事を参照してください。

- [MCP Appsを作成しVS Codeのチャットに自作UIを表示する - Qiita](https://qiita.com/sey323/private/8bb27b8e4ae390c67786)

## セットアップ

```bash
npm install
```

## ビルドと起動

```bash
npm run build
npm start
```
